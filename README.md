# Thomas

A complete tool for organizing your company's support. It controls external calls and guarantees a more efficient in assistance to your users

### Technologies implemented:

- .NET Core 5
- ASP.NET MVC Core
- ASP.NET Identity
- AutoMapper
- Entity Framework Core 3.1
- FluentValidator
- KissLog
